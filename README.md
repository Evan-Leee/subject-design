##项目运行基本步骤：


###step1:

  将项目克隆到本地
  
  `git clone https://github.com/Evan-Leee/homework-search.git`
  
###step2:
 
  在命令行下进入`homework-search/`目录下，然后使用命令
  
  ` npm install `
  
  下载依赖包

###step3:

  输入命令
  
  ` gulp start `
  
  启动服务器，在浏览器中打开
  
  ` localhost:3000 `
  
  查看网页

##测试基本步骤：


###step1:

  在命令行输入
  
  `cucumber features/`
  
  执行测试内容
